NeatJS 🧠

NeatJS is a Javascript implementation of the NeuroEvolution of Augmenting Topologies, which is is a genetic algorithm (GA) for the generation of evolving artificial neural networks (a neuroevolution technique) developed by Ken Stanley.

QuizKNN 📷

A Quiz game using KNN (K-Nearest Neighbors) Image Classification from ML5js. You can set your own examples of what true and false look like using your webcam, once you're done setting up examples you can play the quiz answering the questions with your webcam.

Whiteboard ✏️

This project involves a React frontend connected to a NodeJS Backend using Socket.IO. Anything you write will be seen by anyone who visits the website and will disappear in a few minutes.

ISSTracker 🔭

This is a Real Time 3D tracker of the ISS, developed with Three.js and Satellite.js using NASA's 3D model and trajectory data of the Space Station.

Registro LampSchool 📚
